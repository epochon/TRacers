# Project: Traces - Trajectory-Aware Collective System for Student Retention


## Goal
To create personalized Mentoring Platform for Students


## Problem Statement    
40% of dropouts are caused by administrative friction (scholarship delays, housing issues, language barriers), not academic failure.
Current systems track performance (grades), but they don’t track the hurdles that lead to failure as students feel overwhelmed and unsupported until it’s too late.

## Key Features
- Tracks bureaucratic friction instead of only academic performance
- Domain Agents continuously monitor key student lifelines:
- Financial friction (fees, scholarships)
- Academic irregularities (attendance, sub
- Student can do anonymous chats with their seniors
- Verified peer-to-peer mentoring network
- Matches juniors with seniors based on Course,Batch,Academic Context
